# Challenge 7 - Slicing Figma design into code with React Vite & Tailwindcss

# Description

This is YourLogo company landing page built in ReactJS and TailwindCSS.

# Folder Structure

```
root/
  ├── node_modules/
  ├── public/
  ├── src/
  │ ├── components/
  │ │ ├── container/
  │ │ └── ui/
  │ └── App.tsx
  │ └── index.css
  │ └── main.tsx
  ├── .gitignore
  ├── eslint.config.js
  ├── package-lock.json
  ├── package.json
  └── readme.md
  └── tsconfig.app.json
  └── tsconfig.json
  └── tsconfig.node.json
  └── vite.config.ts
```

# Start the app

Don't forget to run

```
npm install
```

for this project first, then to run the app, run

`npm run dev` and open the page in your browser at `http://localhost:5173`

# Toggle Dark Mode

In `index.html` modify the class of `<html>` with `dark` or `light`.

ex:

```html
<html class="dark"></html>
```
